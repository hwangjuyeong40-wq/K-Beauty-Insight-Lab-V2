# 💄 K-Beauty Insight Lab v2

### 🔍 프로젝트 개요
글로벌 시장 내 K-뷰티 제품의 고객 리뷰 데이터를 분석하여, 배송·포장 품질 및 감성적 만족도를 도출하는 데이터 분석 대시보드입니다.

### 🛠 기술 스택
- **Language:** Python 3.14
- **Framework:** Streamlit
- **Visualization:** Plotly Express
- **Data Analysis:** Pandas
- **Deployment:** Streamlit Community Cloud

### 📊 주요 기능
- **다국어 물류 이슈 비교:** 한국어와 중국어 리뷰 기반 물류 이슈 분포 시각화
- **만족도 지표(KPI) 산출:** 감성 가중치(긍정 1.0, 중립 0.5, 부정 0)를 활용한 품목별 만족도 평가
- **인터랙티브 대시보드:** 필터링 기능(품목, 언어, 감성, 키워드)을 통한 심층 데이터 조회

### 🏗 데이터 파이프라인 설계

* **데이터 수집:** 네이버 리뷰 데이터 및 샤오홍슈 구조화 생성 데이터
* **데이터 정제:** Python Pandas를 활용한 결측치 처리 및 텍스트 데이터 정형화
* **향후 계획:** SQLite를 활용한 관계형 데이터베이스(RDBMS) 연동 및 쿼리 최적화

### 📝 프로젝트를 통해 얻은 인사이트
- **데이터 기반 의사결정:** 정성적인 리뷰 데이터를 정량적 KPI로 전환하여 물류 리스크를 구체화함.
- **도메인 확장성:** 단순 분석을 넘어, 특정 카테고리(K-뷰티)에 특화된 인사이트 도출 가능성 확인.
